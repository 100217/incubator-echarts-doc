# angleAxis(Object)

极坐标系的角度轴。

## polarIndex(number) = 0

角度轴所在的极坐标系的索引，默认使用第一个极坐标系。

## startAngle(number) = 90

其实刻度的角度，默认为 90 度，即圆心的正上方。0 度为圆心的正右方。
